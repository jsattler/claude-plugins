# Workout Templates

Each template includes coaching notes and the workout in Intervals.icu format (ready to paste into the workout builder or add via the API). See `references/intervals-icu-workout-format.md` for full syntax details.

Adapt all templates to the athlete's actual FTP, zones, and available time. The percentages below are starting points — adjust based on the athlete's level and fatigue.

---

## Endurance Rides

### Classic Z2 Endurance

**Purpose:** Aerobic base building, fat oxidation, mitochondrial development.
**RPE:** 3-4. Keep power steady. Resist the urge to push on climbs. Fuel every 45 minutes after the first hour.

```
Warmup
- 10m ramp 50%-65% 90rpm

Endurance
- 160m 60-75% 85-95rpm

Cooldown
- 10m ramp 65%-50% 90rpm
```

(Adjust the 160m endurance block to fit the target ride duration: 2-5 hours.)

### Progressive Endurance

**Purpose:** Build fatigue resistance, simulate late-event pacing. Practice race nutrition strategy.

```
Warmup
- 10m ramp 50%-60% 90rpm

Low Z2
- 100m 60-68% 85-95rpm

Upper Z2 / Low Z3
- 70m 70-80% 85-95rpm

Cooldown
- 10m ramp 65%-50% 90rpm
```

(Adjust block durations to fit 3-4 hours total. The split is roughly 60% low / 40% upper.)

## Sweetspot / Threshold

### Sweetspot Intervals

**Purpose:** Time-efficient fitness building, raise FTP.
**RPE:** 6-7 during intervals.

```
Warmup
- 15m ramp 50%-70% 90rpm

Main Set 3x
- Sweetspot 12m 88-93% 85-95rpm
- Recovery 5m Z1 90rpm

Cooldown
- 10m ramp 60%-45% 90rpm
```

(Scale to 4x for fitter athletes or longer sessions.)

### Over-Under Intervals

**Purpose:** Threshold tolerance, ability to recover from surges at threshold.
**RPE:** 7-8 on the overs.

```
Warmup
- 15m ramp 50%-75% 90rpm

Main Set 3x
- Over 1m 105% 90rpm
- Under 2m 95% 90rpm
- Over 1m 105% 90rpm
- Under 2m 95% 90rpm
- Over 1m 105% 90rpm
- Under 2m 95% 90rpm
- Over 1m 105% 90rpm
- Under 2m 95% 90rpm
- Recovery 6m Z1 90rpm

Cooldown
- 10m ramp 60%-45% 90rpm
```

### Threshold Cruise Intervals

**Purpose:** Extend time at threshold, build sustainable power.
**RPE:** 7-8.

```
Warmup
- 15m ramp 50%-75% 90rpm

Main Set 3x
- Threshold 18m 95-100% 90rpm
- Recovery 6m Z1 90rpm

Cooldown
- 10m ramp 60%-45% 90rpm
```

## VO2max

### Classic VO2max Intervals

**Purpose:** Increase VO2max ceiling, improve oxygen delivery.
**RPE:** 9 during intervals. These should feel hard but sustainable for the interval duration.

```
Warmup
- 15m ramp 50%-70% 90rpm

Main Set 5x
- VO2max 4m 108-115% 95-105rpm
- Recovery 4m Z1 85rpm

Cooldown
- 10m ramp 55%-40% 85rpm
```

### Hill Repeats

**Purpose:** Climbing-specific power, neuromuscular recruitment.
**RPE:** 8-9 on the climbs. Find a sustained climb of 3-5 minutes.

```
Warmup
- 15m ramp 50%-70% 90rpm
- Opener 1m 100% 95rpm
- 2m Z1 90rpm
- Opener 1m 100% 95rpm
- 2m Z1 90rpm

Main Set 6x
- Climb 4m 105-115% 70-80rpm
- Descend/recover 4m Z1 90rpm

Cooldown
- 10m ramp 55%-40% 85rpm
```

## Recovery

### Active Recovery Ride

**Purpose:** Promote blood flow without adding training stress.
**RPE:** 1-2. Truly easy. If in doubt, go easier.

```
Recovery
- 45m Z1 90-100rpm
```

## Event-Specific

### Race/Event Simulation

**Purpose:** Full dress rehearsal 2-3 weeks before the target event. Practice nutrition, pacing, position changes.

```
Warmup
- 15m ramp 50%-65% 90rpm

Steady riding
- 60m 60-70% 85-95rpm

Tempo climb simulation 3x
- Climb 15m 80-88% 75-85rpm
- Recover 10m 60-65% 90rpm

Endurance finish
- 60m 65-72% 85-95rpm

Cooldown
- 10m ramp 60%-45% 90rpm
```

(Adjust block durations to match the target event profile and duration.)

### Low Cadence Force Work

**Purpose:** Muscular endurance for long climbs, reduced cardiovascular demand at given power.
**RPE:** 7. Focus on smooth pedal stroke at low cadence.

```
Warmup
- 15m ramp 50%-75% 90rpm

Main Set 6x
- Low cadence 5m 85-90% 55-65rpm
- Spin-up 5m Z1 100rpm

Cooldown
- 10m ramp 60%-45% 90rpm
```

### Climbing Endurance

**Purpose:** Build climbing-specific endurance and pacing discipline.
**Notes:** Pick a route with sustained climbing (1500-2500m elevation gain). This is a feel-based ride — targets are guidelines.

```
Warmup
- 15m ramp 50%-65% 90rpm

Climbing
- 150m 65-80% 75-90rpm

Cooldown
- 15m ramp 60%-45% 90rpm
```

(The climbing block is intentionally broad — power will vary with terrain. Aim to keep climbing efforts at 75-88% and recover on descents.)
